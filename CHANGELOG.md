# Revision history for Spindle

## 0.1.0.0 -- YYYY-mm-dd

Spindle 0.1.1.0 Goal (recommended)

“A small functional language with:

- integers, booleans
- if
- lambdas + application
- let with Hindley–Milner type inference
- an interpreter
- a tiny standard library baked in (like add, sub, eq)”

* Out of scope for 0.1:
- modules
- typeclasses
- dependent types
- effects / IO
- LLVM
- optimizations beyond constant folding
